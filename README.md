 # 🧳 미래투어
 
 렌터카, 숙박, 티켓을 판매하는 국내여행 전문 웹사이트<br><br>
 
 
 ## 주요 기능
 - 회원 : 회원 가입, 마이페이지(정보 수정, 주문 내역 확인, 장바구니), 상품 검색 및 결제, 공지사항 읽기, 1대1 문의 작성하기
 - 관리자 : 상품 등록 및 관리, 회원 리스트 보기, 공지사항 작성 및 등록된 문의에 답변하기<br><br>
